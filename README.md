# 📖 About

CareerPilot AI is an intelligent web application built using the **MERN Stack** and **Google Gemini AI** that analyzes a candidate's resume against a job description.

The platform evaluates resume content, identifies matching and missing skills, estimates an ATS-style compatibility score, and generates personalized suggestions to improve interview readiness. It helps job seekers understand how well their profile aligns with specific job requirements and provides actionable insights to strengthen their resumes.

# ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/PrachiPujari10/CareerPilot-AI-Intelligent-Interview-Resume-Analyzer.git
```

### Install Backend Dependencies

```bash
cd backend
npm install
npm start
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

### Backend (.env)

```env
MONGODB_URI=

GEMINI_API_KEY=

PORT=5000
```

### Frontend (.env)
